# Make My QR 

This version is the sync-enabled and backend build. It keeps accounts, profile details, and QR history in the server database and does require a local server.

## Run it

1. Install Node.js 18 or newer.
2. Open this folder in a terminal.
3. Start the app with `npm start`.
4. Open the printed URL in your browser.

## Use it from another device

1. Start the app on the main machine.
2. Find the app address printed in the terminal.
3. Open that same address from the other device while both devices are on the same network.
4. Log in with the same account to see the same saved history.
